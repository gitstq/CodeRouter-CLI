# CodeRouter-CLI

Lightweight Terminal AI Model Intelligent Routing Engine.

## Installation

```bash
pip install -e .
```

## Usage

```bash
coderouter serve          # Start API proxy server
coderouter config list    # List configured models
coderouter config add     # Add a model
coderouter stats          # View cost statistics
coderouter test           # Test model connectivity
coderouter tui            # Launch TUI interface
```
